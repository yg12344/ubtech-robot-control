# 优必选小微机器人控制软件 - 项目概述

## 项目完成情况

已成功创建一个功能完整的优必选小微机器人综合控制软件，采用PyQt6桌面应用技术栈。

## 已实现功能

### ✅ 核心架构
- **主界面框架**: 侧边栏导航 + 多功能内容区
- **通信接口层**: 支持串口和TCP/IP两种连接方式
- **模块化设计**: 清晰的代码结构，易于扩展

### ✅ 五大功能模块

1. **运动控制模块** (motion_control.py)
   - 基础动作：站立、坐下、行走、停止
   - 手势动作：挥手、点头、摇头、鞠躬
   - 行走控制：方向、距离、速度调节
   - 舞蹈动作：多种预设舞蹈
   - 紧急停止功能

2. **语音交互模块** (voice_control.py)
   - 语音识别（支持中文/英文）
   - 语音合成（让机器人说话）
   - 麦克风选择和语言配置
   - 预设语音指令快速调用

3. **编程教学模块** (code_editor.py)
   - Python代码编辑器
   - 语法高亮显示
   - 丰富的示例代码（基础/进阶/高级）
   - 代码运行和输出显示

4. **远程监控模块** (monitor.py)
   - 实时机器人状态监控
   - 传感器数据显示
   - 数据曲线图表（matplotlib）
   - 电池、温度、距离等数据可视化

5. **系统设置模块** (settings.py)
   - 连接方式配置（串口/TCP）
   - 语音识别设置
   - 常规选项配置
   - 配置保存与导入

## 项目结构

```
优必选小微控制软件/
├── main.py                 # 程序入口
├── robot_interface.py      # 机器人通信接口层

├── ui/                    # UI模块
│   ├── __init__.py

    ├── __init__.py

    ├── voice_control.py   # 语音交互

    ├── monitor.py         # 远程监控

```

## 技术栈

- **GUI框架**: PyQt6
- **图表绘制**: matplotlib
- **语音识别**: SpeechRecognition + pyaudio
- **串口通信**: pyserial
- **数据格式**: JSON

## 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行程序
```bash
python main.py
```

## 使用流程

1. **配置连接**: 在"系统设置"中选择连接方式（串口或TCP/IP）
2. **连接机器人**: 点击侧边栏"连接机器人"按钮
3. **选择功能**: 通过侧边栏导航切换不同功能模块
4. **执行控制**: 在各功能页面发送指令或操作

## 后续建议

由于您有硬件但不了解具体型号，建议：

1. **确认机器人型号**: 查看机器人标识，确定具体型号
2. **获取API文档**: 联系优必选官方获取对应型号的SDK或API文档
3. **调整通信协议**: 根据实际API修改 `robot_interface.py` 中的指令格式
4. **测试连接**: 先测试基本的连接和指令发送
5. **扩展功能**: 根据机器人实际能力添加更多动作和功能

## 注意事项

- 当前代码中的通信指令格式为通用格式，实际使用时需根据机器人API调整
- 语音识别功能需要配置麦克风
- 数据监控模块目前显示模拟数据，连接实际机器人后可显示真实数据

## 技术亮点

1. **模块化设计**: 各功能模块独立，易于维护和扩展
2. **异步通信**: 使用独立线程处理数据接收，不阻塞UI
3. **配置持久化**: 支持保存和加载配置
4. **实时数据监控**: 使用matplotlib绘制实时数据曲线
5. **友好UI**: 现代化的界面设计，操作直观
