# 优必选小微机器人控制软件 - Android版本

一个功能完整的Android应用，用于控制优必选小微机器人。

## 主要特性

### 🎮 运动控制
- 基础动作：站立、坐下、行走、停止
- 手势动作：挥手、点头、摇头、鞠躬
- 行走控制：方向、距离、速度调节
- 多种舞蹈动作
- 紧急停止

### 🎤 语音交互
- 语音识别（支持中文和英文）
- 语音合成（让机器人说话）
- 快速语音指令
- 语言选择

### 💻 编程教学
- Python代码编辑器
- 丰富示例代码（基础/进阶/高级）
- 代码执行功能
- 输出显示

### 📊 远程监控
- 实时状态监控
- 传感器数据显示
- 数据趋势图
- 电池、温度等数据

### ⚙️ 系统设置
- 蓝牙连接配置
- TCP/IP连接配置
- 语音识别设置
- 配置保存与加载

## 安装方法

### 方法1: 直接安装APK

1. 下载 `ubtech_robot_control-1.0.0-release.apk`
2. 在Android设备上打开APK文件
3. 按照提示完成安装
4. 首次运行时授予所需权限

### 方法2: 从源码编译

参考 `打包说明.md` 了解如何从源码打包APK。

## 使用指南

### 首次使用

1. **授予权限**
   - 启动应用后，授予蓝牙、网络、录音等权限

2. **配置连接**
   - 进入"系统设置"
   - 选择连接方式（蓝牙或TCP/IP）
   - 输入相应的连接参数
   - 点击"保存"

3. **连接机器人**
   - 返回主界面
   - 点击"连接机器人"按钮
   - 等待连接成功

### 功能使用

#### 运动控制
1. 点击"运动控制"卡片
2. 选择要执行的动作
3. 点击按钮发送指令

#### 语音交互
1. 点击"语音交互"卡片
2. 点击"开始监听"进行语音识别
3. 输入文本让机器人说话
4. 使用快速指令

#### 编程学习
1. 点击"编程教学"卡片
2. 选择示例代码
3. 在编辑器中查看和修改
4. 点击"运行"执行

#### 数据监控
1. 点击"数据监控"卡片
2. 点击"开始监控"
3. 查看实时数据

## 连接方式

### 蓝牙连接
1. 在设置中选择"蓝牙连接"
2. 输入机器人的MAC地址
3. 保存后连接

### TCP/IP连接
1. 在设置中选择"TCP/IP连接"
2. 输入机器人的IP地址和端口
3. 保存后连接

## 权限说明

应用需要以下权限：
- **蓝牙**: 用于连接蓝牙机器人
- **网络**: 用于TCP/IP连接
- **录音**: 用于语音识别
- **网络状态**: 检测网络连接

## 系统要求

- Android 5.0 (API 21) 或更高版本
- 支持蓝牙4.0或更高版本
- 建议至少2GB RAM
- 存储空间：至少50MB

## 常见问题

### Q: 无法连接到机器人？
A:
- 检查连接方式配置是否正确
- 确认机器人已开机
- 检查蓝牙配对状态
- 检查网络连接（TCP模式）

### Q: 语音识别不准确？
A:
- 确保已授予录音权限
- 在安静环境中使用
- 调整麦克风位置
- 尝试切换识别语言

### Q: 应用闪退？
A:
- 确认Android版本是否满足要求
- 清除应用数据后重试
- 卸载后重新安装
- 联系技术支持

### Q: 如何获取机器人MAC地址？
A:
- 进入手机蓝牙设置
- 查找已配对的设备
- 点击设备详情查看MAC地址
