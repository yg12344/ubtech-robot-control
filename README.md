# 优必选小微机器人控制软件

一个综合性的桌面控制平台，支持优必选小微机器人的多种控制方式。

## 功能特性

### 🎮 运动控制
- 基础动作：站立、坐下、行走、停止
- 手势动作：挥手、点头、摇头、鞠躬
- 行走控制：方向、距离、速度调节
- 舞蹈动作：多种预设舞蹈
- 紧急停止功能

### 🎤 语音交互
- 语音识别（支持中文和英文）
- 语音合成（机器人说话）
- 麦克风选择和语言设置
- 预设语音指令

### 💻 编程教学
- Python代码编辑器
- 语法高亮显示
- 丰富的示例代码（基础/进阶/高级）
- 代码执行和输出显示

### 📊 远程监控
- 实时机器人状态监控
- 传感器数据显示
- 数据曲线图表
- 电池、温度、距离等数据可视化

### ⚙️ 系统设置
- 多种连接方式：串口、TCP/IP
- 语音识别配置
- 自动连接等常规设置
- 配置保存和导入

## 安装说明

### 环境要求
- Python 3.8+
- Windows / macOS / Linux

### 安装步骤

1. 克隆或下载项目
```bash
cd 优必选小微控制软件
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行程序
```bash
python main.py
```

## 使用指南

### 首次使用

1. **连接机器人**
   - 点击侧边栏的"连接机器人"按钮
   - 在"系统设置"中配置连接方式（串口或TCP/IP）
   - 保存配置后重新连接

2. **运动控制**
   - 进入"运动控制"页面
   - 点击按钮发送动作指令
   - 使用滑块调节行走速度

3. **语音交互**
   - 进入"语音交互"页面
   - 选择麦克风和识别语言
   - 点击"开始监听"进行语音识别
   - 输入文本让机器人说话

4. **编程学习**
   - 进入"编程教学"页面
   - 从左侧选择示例代码
   - 在代码编辑器中修改
   - 点击"运行代码"执行

5. **数据监控**
   - 进入"数据监控"页面
   - 点击"开始监控"
   - 查看实时状态和数据曲线

## 通信接口

### 串口连接
- 默认波特率：115200
- 支持的串口：自动检测可用串口

### TCP/IP连接
- 默认主机：192.168.1.100
- 默认端口：8080
- 协议：JSON格式数据包

### 指令格式
```json
{
  "type": "motion|speech|code|query",
  "action": "动作名称",
  "timestamp": 时间戳,
  "其他参数": "..."
}
```

## 配置说明

配置文件 `config.json` 包含以下部分：

- **connection**: 连接设置（串口/TCP）
- **voice**: 语音识别和合成设置
- **general**: 常规设置（自动连接、更新间隔等）

## 注意事项

1. **硬件要求**
   - 需要优必选小微机器人硬件
   - 支持串口或网络连接

2. **依赖说明**

3. **平台兼容性**

   - macOS: 完全支持
   - Linux: 完全支持

## 常见问题

### Q: 无法连接到机器人？
A:
- 检查连接方式和参数配置
- 确认机器人已开机
- 检查串口线或网络连接

### Q: 语音识别不准确？
A:
- 确保麦克风工作正常
- 在安静环境中使用
- 调整麦克风音量
- 尝试切换识别语言

### Q: 代码执行无响应？
A:
- 确认机器人已连接
- 检查代码语法是否正确
- 部分功能需要机器人API支持
